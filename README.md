# 🐍 Python Mini Projects

A collection of small Python projects built while learning core programming concepts — logic building, functions, and simple I/O handling.

## 📂 Projects

| File | Description |
|---|---|
| `simple_cal.py` | A basic calculator supporting core arithmetic operations |
| `rent_cal.py` | Calculates rent split/shares based on user input |
| `tic_tac_toe.py` | Console-based Tic-Tac-Toe game for two players |
| `simple_notepad.py` | A basic notepad-style text editor |
| `game.py` | A simple interactive game built for practice |

## 🛠️ Tech Stack
- Python (core, no external libraries)

## 🚀 How to Run
```bash
python <filename>.py
```

## 🎯 Purpose
These were built to strengthen fundamentals — loops, conditionals, functions, and basic user interaction — before moving into AI/ML projects.
